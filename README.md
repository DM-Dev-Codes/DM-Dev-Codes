## 📂 Projects

### WhatsApp Invoice Processor
A microservice platform that lets users snap a photo of an invoice and send it via WhatsApp.  
It extracts structured data with GPT, stores records in AWS RDS, and lets you ask financial questions in plain English.  
Built with Python · FastAPI · Docker · Kafka · Redis · Twilio · OpenAI · AWS (S3, RDS, EC2)

### Mass Messaging Automation
A Python tool for coordinating message dispatch across WhatsApp, Telegram and X.  
Supports text, image and video content with asynchronous queues and MySQL-based session tracking.  
Built with Python · asyncio · REST API · MySQL · Docker
